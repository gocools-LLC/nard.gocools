# Nard v0.1.0 Release Notes (2026-03-05)

Initial public MVP release for `nard.gocools`.

## Highlights

- Core protocol draft (`RFC-0002`) and peer discovery foundations.
- Decentralized naming registry and QUIC transport abstraction.
- Edge node agent skeleton with node capability/state APIs.
- Heartbeat-driven routing with unhealthy-peer failover decisions.
- IPv6-first and dual-stack connectivity tests with dedicated CI target.
- Node identity documents with key rotation/revocation model.
- CLI lifecycle commands: `nard node start|join|status`.
- Devnet smoke test script and v0.1.0 release gating checklist.
